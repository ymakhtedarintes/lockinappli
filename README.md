# Lock In — Habit Tracker

A personal productivity web app built to track daily habits, fitness goals, and screen time. Designed to be used daily as a mobile-first PWA.

**Live:** [lockinappli.vercel.app](https://lockinappli.vercel.app)

---

## Features

- Daily habit checklist with streak tracking
- Weekly run logger with progress bar
- Social media countdown timer with a configurable daily cap
- 7-day completion bar chart
- Countdown to a target goal date
- Saves data locally — no account needed
- Installable as a PWA on iOS and Android

## Tech Stack

- Next.js 14 (App Router)
- React 18
- Recharts
- Deployed on Vercel

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Motivation

Built this because I wanted a single place to track everything (gym, reading, studying, running, and screen time) without paying for an app or dealing with bloated features. Kept it minimal and fast on purpose.
